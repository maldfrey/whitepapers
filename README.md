# Maldfrey Whitepapers

A collection of essays exploring civilizational dynamics, moral toolkits, institutional failure, and cultural ratchets.

### Philosophy

These papers form a connected body of work examining why complex societies repeatedly fail to solve solvable problems. The central thesis is that civilizations require **dynamic complementarity** between two distinct moral-strategic toolkits (Mode A and Mode B). When one mode dominates without correction, predictable patterns of fragility, institutional paralysis, and self-undermining behavior emerge.

### Recommended Reading Order
1. **The Third Factor Heuristic** – Best starting point (short & practical)  
2. **The Mode B Ratchet** – Core diagnostic  
3. **Conceit-Anchored Nihilism** – Psychological layer

6. **Anti-Natalism: The Poisoned Well of a Mode B Monoculture** – Latest application
